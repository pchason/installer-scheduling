# AGENTS.md

## Setup commands

- Install deps: `npm install`
- Start dev server: `npm dev`
- Run tests: `npm test`

## Code style

- TypeScript strict mode
- Single quotes
- Use functional patterns where possible

## Security

- Do not read the contents of .env files
- Do not allow cross-site scripting, prompt injection, unpermitted access or any other security issues that OWASP's web application security threates list
